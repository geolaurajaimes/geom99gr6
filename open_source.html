KoBoToolbox

KoBoToolbox is an open source platform for data collection in the field. Allows organizations to create surveys, forms and collect data on mobile devices without an internet connection.

Advantages

  • Easy to use
  • Works without internet connection
  • Customizable and user - friendly

Disadvantages

  • Limited free

qFields

qFields is an open source platform similar use case as FieldMaps it allows for data collection and customization as well as editing exsiting data. it has the capability to support standalone datasets like vector formats, (KML, GeoJSON, Shapefile, and Geopackage datasets. as well as suporting rasters, (GeoTIFF, JPEG, PNG etc.) these data sets re imported from a URL or compressed peoject file. this application can be run on mobile devoces IOS and Android and also on desktops. Utilizing the QGIS server to support the software.

This solution allows the collection and evaluation of trees.

Advantages

  • Accessible and automated platform

Disadvantages

  • It is recommended for small areas at the neighborhood level.
